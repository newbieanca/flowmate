🔥 Koneksi Firebase (Realtime Database)
⬆️ Paste seluruh objek config dari Firebase Console. Semua perangkat yang menggunakan config yang sama akan tersinkron secara realtime.
✅ Tersimpan!
📋 Cara Setup Firebase (5 menit)
1
Buka console.firebase.google.com → Create a project (gratis, pilih nama bebas)
2
Di project baru, klik Realtime Database → Create database → pilih region asia-southeast1 → pilih mode Test mode dulu (bisa diubah nanti)
3
Klik ⚙️ Project Settings → scroll ke Your apps → klik </> Web → register app → copy bagian firebaseConfig = { ... }
4
Paste config JSON tersebut ke kolom di atas, isi nama operator, klik Simpan & Hubungkan
5
Bagikan file HTML ini + config yang sama ke semua device. Semua akan terhubung dan update realtime secara otomatis ✅
💡 Free tier Firebase cukup untuk penggunaan factory floor: 1 GB storage, 10 GB/bulan transfer, 100 koneksi simultan.
