⚙️ Pengaturan
🔥 Firebase Realtime Database
Paste seluruh objek config dari Firebase Console → Project Settings → Your Apps → Web.
💡 Free tier Firebase cukup untuk factory floor: 1 GB storage, 10 GB/bulan transfer, 100 koneksi simultan.
📊 Google Apps Script – Sheets Mirror & BOM

GAS digunakan untuk 2 fungsi: (1) menyalin transaksi dari Firebase ke Google Sheets secara otomatis untuk keperluan tracing, dan (2) mengambil BOM saat input Output Produksi. Wajib diisi jika ingin kedua fitur ini aktif.

📋 Cara Deploy GAS (3 langkah)
1
Buka Google Sheets database FLOWMATE → Extensions → Apps Script → paste isi file flowmate_GAS.gs → Save
2
Klik Deploy → New deployment → Type: Web App → Execute as: Me → Who has access: Anyone → Deploy
3
Copy URL deployment yang muncul → paste ke kolom di atas → klik Simpan URL GAS → klik Test Koneksi GAS untuk verifikasi
⚠️ Setiap kali script GAS diupdate, buat deployment baru (bukan edit yang lama) agar perubahan aktif.
📋 Cara Setup Firebase (5 menit)
1
Buka console.firebase.google.com → Create a project (gratis, pilih nama bebas)
2
Klik Realtime Database → Create database → region asia-southeast1 → mode Test mode
3
⚙️ Project Settings → Your apps → klik </> Web → register → copy firebaseConfig = { ... }
4
Paste config di atas, isi nama operator → klik Simpan & Hubungkan Firebase. Badge hijau = berhasil.
